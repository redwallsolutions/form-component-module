## Introduction

Form Component Module is a react component package that wraps up [informed](https://github.com/joepuzzo/informed) mechanism and
applies some material design style to input and textarea components.

![demogif](src/lib/assets/demo.gif)

## Installation

You can install it using `yarn`:

    yarn add @redwallsolutions/form-component-module

Or using `npm`:

    npm i @redwallsolutions/form-component-module

### Available Components

	import {InputField, TextAreaField, Form} from '@redwallsolution/form-component-module'

## Usage

### Simplest Form

<Playground>
	<InputField field="somename" label="My Input" />
</Playground>

### The shouldFitContainer prop 

It's possible to the field fit to parent width.

<Playground>
	<InputField field="fitmyparent" label="I'm responsible" shouldFitContainer/>
</Playground>

### Password Type

<Playground>
	<InputField field="password" label="What's your password?" type="password" />
</Playground>

### With informed form
<Playground>
	<Form onSubmit={console.log}>
		<InputField field="somefield" label="Some Field" />
		<InputField field="OtherField" label="Other Field" />
        <button type="submit">Save</button>
        <div><small><em>Pro tip: After save it, look at console!</em></small></div>
	</Form>
</Playground>


### TextAreaField or isMultiline prop

You can use the `TextAreaField` component to an alternative for long texts.

<Playground>
	<TextAreaField label="It's a place for type a loooong text." field="mylongtext" shouldFitContainer/>
</Playground>

You also can force a `InputField` component to behave itself as a textarea. Just pass the `isMultiline` prop to it. 

<Playground>
	<InputField label="I'm pretending to be bigger." field="faketextarea" isMultiline/>
</Playground>
